GUITAR SWITCHER

Guitar switcher thought for an on-stage use where multiple guitars are use. The user sets the cut or boost for each channel and select it with the footswitch (indicated by an LED). Considering that different guitars may have differents output level (lespaul compared to a telecaster) this little thing could make the rig consistent avec sound level and other pedal.

the number of output used is set via dipswitch. The switching is made with a CD4017 counter, the muting system used a CD4066 analog switcher. A second switching system could be added to mute the output and route the signal to a second output for a tuner.

The signal should be kept low to avoid distortion from the CD4066.
