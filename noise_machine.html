NOISE MACHINE

Idea for a noise machine controlled by joystick. Although the board was build, the all thing wasn't built.

I might re-think this project that still could be interesting with these joysticks. Maybe add a bit of digital in it. It won't be what most people expects from a musical instrument, but it could be fun nonetheless.



The circuit was using very simple CMOS ICs, op-amp and OTA. The joysticks were supposed to control two key elements for each. The user could select the destination of the modulation and mix the oscilators with noise.



The only real issue was to make an efficient and simple VCA without using (again) an OTA (although I'll probably do that next time) and adding too much noise and bleedthrough. Each sides (oscillator and noise) had their own filter selectable between Lowpass and banpass.

This project led to the Wintermute project.
